> Sextou seus aligenígena!

## Compatibility

- Linux 
- Windows
- Mac

## Installing
Install globally:

    npm install -g sextou

## Running
Run command:

    sextou

Or import in NodeJS script:

    import sextou from 'sextou';

    sextou();
    
## Running with [npx](https://www.npmjs.com/package/npx)
Run command:

    npx sextou
